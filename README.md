# IDT-game

## Overview
본 프로젝트는 고전 플래시게임 [Inuyasha Demon Tournament]를 모티브로 삼아 진행된 온라인 턴제 멀티플레이어 게임입니다. 컴퓨터를 상대로 플레이하는 해당 게임과는 다르게, 웹 환경에서 실시간으로 다른 유저와 대결하는 것을 목표로 개발되었습니다.

## Play Game
https://idt-game.herokuapp.com/
